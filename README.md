# Fixify: AI-Powered Home Maintenance Application

## Executive Summary
This repository contains the comprehensive project management documentation and business planning for **Fixify**, an innovative AI-powered mobile application. Designed to revolutionize the home maintenance industry in Saudi Arabia, Fixify leverages artificial intelligence to provide homeowners with real-time issue diagnosis, cost estimation, and seamless booking with certified technicians. 

## Core Objectives
- **Innovative Problem Solving:** Conceptualize a Live AI Repair Assistant that utilizes smartphone cameras to analyze household damage and predict repair costs instantly.
- **End-to-End Project Planning:** Apply PMBOK frameworks to manage the project lifecycle across five phases: Initiating, Planning, Executing, Monitoring & Controlling, and Closing.
- **Strategic Business Alignment:** Develop robust operational, marketing, and financial plans to ensure market viability, targeting a 131% Return on Investment (ROI) over a 12-month deployment timeline.

## Project Management Methodology & Tools
- **Framework:** Executed strictly following the Project Management Body of Knowledge (PMBOK) guidelines.
- **Scheduling & Tracking:** Utilized **GanttPro** to build a dynamic, 12-month network diagram and Work Breakdown Structure (WBS) to track task dependencies and actual-versus-estimated time.
- **Task Delegation:** Deployed **Trello** using Kanban boards to assign, monitor, and control sprint tasks across the project management lifecycle.
- **Risk Management:** Developed a comprehensive Probability/Impact matrix and Risk Register to proactively mitigate technical delays and budget overruns.

## Key Features of Fixify
- **Live AI Repair Assistant:** Real-time troubleshooting and fault detection via computer vision.
- **Automated Cost Estimation:** Machine learning algorithms that predict service costs prior to technician deployment.
- **Technician Marketplace:** A vetted, secure network for booking scheduling, and reviewing professional maintenance services.
- **Predictive Maintenance:** HVAC monitoring and emergency alert integration.
